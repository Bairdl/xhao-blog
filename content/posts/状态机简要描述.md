**状态机 = 状态 + 事件 + 转换规则**

// 比如Session状态机：
状态：未认证、认证中、已认证、关闭中、已关闭

事件：登录请求、登录成功、登录失败、退出、强制关闭

转换规则：
- 未认证 + 登录请求 → 认证中
- 认证中 + 登录成功 → 已认证  
- 认证中 + 登录失败 → 未认证
- 任何状态 + 强制关闭 → 已关闭

核心思想：明确的状态和清晰的转换条件，避免非法状态出现。
